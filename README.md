# World Cup 2026 Predictor

Machine learning powered FIFA World Cup 2026 prediction engine using:

- Dynamic Elo Ratings
- Tournament-weighted Elo updates
- Historical international match data
- Expected Goals (xG)
- Poisson score simulation
- Monte Carlo tournament simulation
- Static HTML reporting

## Features

### Match Prediction

The predictor estimates:

- Home win probability
- Draw probability
- Away win probability
- Expected goals (xG)

### Group Stage Simulation

The engine simulates the complete group stage thousands of times and calculates:

- Expected points
- Probability of finishing 1st
- Probability of finishing 2nd
- Probability of finishing 3rd
- Probability of finishing 4th
- Probability of direct qualification
- Probability of qualifying as one of the best third-placed teams

### Knockout Stage Simulation

The complete FIFA World Cup is simulated using Monte Carlo methods.

Calculated probabilities include:

- Round of 32
- Round of 16
- Quarter-finals
- Semi-finals
- Final
- World Champion

### Reporting

The project automatically generates:

- CSV outputs
- Historical snapshots
- Static HTML dashboards
- English / German report support

---

## Methodology

The prediction pipeline works as follows:

1. Load historical international football results.
2. Calculate dynamic Elo ratings for all teams.
3. Build rolling form statistics.
4. Generate machine learning features.
5. Train HistGradientBoostingRegressor models for expected goals.
6. Convert xG values into Poisson score distributions.
7. Simulate individual matches.
8. Simulate complete World Cup tournaments thousands of times.
9. Aggregate probabilities and generate reports.


Generate the static HTML report:

